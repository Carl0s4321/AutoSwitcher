# AutoSwitcher
Auto tab switcher using ultrasonic sensor

## Requirements
1. Ultra sonic sensor
2. arduino

## Steps
1. py -m pip install pyserial pyautogui
2. Change the Serial port to whatever port your arduino is connnected to
3. build the .cpp file
4. run `python ./src/listener.py`
